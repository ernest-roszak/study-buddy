## Technologies

Project is created with:

- React version: 17.0.1
- React-router-dom version: 5.2.0
- styled-components version: 5.2.1
- axios version: 0.21.1

## Setup

To run this project, install it locally using npm:

```
$ cd study-buddy
$ npm install
$ npm start
```
